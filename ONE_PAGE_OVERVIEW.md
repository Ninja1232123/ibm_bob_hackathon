# 🤖 Bob - One Page Overview

## What Is Bob?
**An AI development companion that learns from your debugging and automates your workflow.**

## The Core Innovation
```
Your Manual Fix → Bob Extracts Pattern → Saves to Database → 
Next Time: Bob Detects Automatically → Suggests/Applies Fix
```

## See It In 2 Minutes
```bash
cd Dev_Master/bob_unified
python -m bob.demo.comprehensive_demo
```
